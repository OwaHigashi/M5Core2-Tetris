# M5Core2-Tetris
## M5Core2ライブラリプロジェクト
M5Core2のSD-Updaterを用いたライブラリ集を構築するプロジェクトの一環として実装されています。

## どんなゲーム

いわずと知れたテトリスゲームです。オリジナルは音が鳴りますが、M5Core2に移植するにあたり、音は再現していません。
